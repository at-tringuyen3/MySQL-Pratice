### Câu 23. Lấy 2 blog, 2 news mà user có id = 1 đã comment
```
(SELECT comment.id, comment.comment, blog.title, blog.content
FROM comment
INNER JOIN blog
WHERE comment.target_table = 'blog' AND comment.user_id = 1 GROUP BY comment.id
ORDER BY comment.id DESC
LIMIT 2)
UNION ALL
(SELECT comment.id, comment.comment, blog.title, blog.content
FROM comment
INNER JOIN blog
WHERE comment.target_table = 'news' AND comment.user_id = 1 GROUP BY comment.id
ORDER BY comment.id DESC
LIMIT 2)
```
