### Câu 19. Lấy 5 blog và 5 news của 1 category bất kỳ
```
SET @id_category = (SELECT id FROM category ORDER BY RAND() LIMIT 1);
(SELECT blog.category_id, blog.title, blog.content FROM blog INNER JOIN category ON blog.category_id = category.id WHERE blog.category_id = @id_category ORDER BY blog.id DESC LIMIT 5)
UNION ALL
(SELECT news.category_id, news.title, news.content FROM news INNER JOIN category ON news.category_id = category.id WHERE news.category_id = @id_category ORDER BY news.id DESC LIMIT 5)
```
